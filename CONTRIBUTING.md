# Contributing to sandworm

Thank you for your interest in contributing to Lineage Foundation **sandworm** (Keccak-prime / PoW hashing library).

## How to contribute

1. **Open an issue** – Bugs, features, or questions in this repository.
2. **Fork and clone** – Fork and clone locally.
3. **Create a branch** – e.g. `feat/...`, `fix/...`.
4. **Make changes** – Follow [.editorconfig](.editorconfig); run `cargo fmt` and `cargo test`.
5. **Submit a PR** – Against `main`.

## Code style

- Rust: 4-space indent per [.editorconfig](.editorconfig).
- [Conventional Commits](https://www.conventionalcommits.org/) for messages.
